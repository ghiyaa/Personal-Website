PORTFOLIO

Jerani

Jerani is a social application based on the website "Nextdoor.com". It's built using Flutter as the framework and Firebase as the backend. It features several pages with different topics of discussion, kind of like Facebook Pages. Once the user signs in, they are asked to select the neighborhood they live in, which will then take them to the discussion posts of that neighborhood. Once signed in, they could post a text, picture or video on that page, like and comment, as well as report a post and delete they comment/post.

Warehouse Management Application (WMA)

WMA is an Android application built in Java and XML, backend built using Firebase. As the name entails, it's a warehouse managing application made for small businesses to keep track of their inventories. Once the user has created their account, they can add and delete products, see what's low in stock, order more, and order new products. The app keeps track of all the warehouse activities and product status.

Emergency Alert App

Emergency Alerting App for the general public. The technology used was Flutter integrated with Google Maps (frontend) and Firebase (backend). The main objective of this app is to post alerts and emergencies. The app has 2 views, Alerts are posted in the form of items on a list (listview) and pins on google maps (mapview), the user can select the Alert they would like to know more of and a detailed alert post would pop up. This app also includes emergency numbers, as well a civil defense bases locations.
